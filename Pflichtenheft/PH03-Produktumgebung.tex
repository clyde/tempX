\section{Produktumgebung}

Das Programm läuft auf einem der Poolrechner im Raum 356 des Informatik Gebäudes 50.34 des Karlsruher Institut of Technologies.

\begin{itemize}
\item Software
	\begin{itemize}
	\item Betriebssystem: 
		\begin{itemize}
			\item Windows 2000/XP/Vista/7
			\item Linux
			\item (optional) Mac OS X 10.5
		\end{itemize}
	
	\item Laufzeitumgebung:
		\begin{itemize}
			\item Java 1.6
		\end{itemize}		
	\end{itemize}
	
\item Hardware 
	\begin{itemize}
	\item Mindestanforderung an den Arbeitsplatzrechner: 
		\begin{itemize}
			\item Dual Core 2 Ghz
			\item 2 GB RAM
			\item Bildschirm mit einer Auflösung von 800 x 600 Pixel
			\item Hardbwarebeschleunigte 3D Unterstützung
		\end{itemize}
	
	\item Kamera:
		\begin{itemize}
			\item Alle Kameramodelle die mindestens den JEITA Exif Version 2.1 Standard vom 1. Juni 1998 einhalten
		\end{itemize}		
	\end{itemize}	
\end{itemize}
